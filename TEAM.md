# AMD 双卡调度器 - 项目团队

## 项目概述
**产品**: AMD 双卡 GPU 任务调度系统
**技术栈**: Go 1.26.1 + ROCm/OpenCL + gRPC
**目标**: 深度学习训练、GPU 计算任务的智能调度

## 团队成员

| Agent ID | 角色 | 职责 |
|----------|------|------|
| `amd-team-leader` | 团队负责人 | 协调、决策、进度跟踪 |
| `amd-pm` | 产品经理 | 需求分析、PRD、调度策略设计 |
| `amd-arch` | 技术架构师 | 系统设计、GPU 通信架构 |
| `amd-backend` | 后端工程师 | Go 实现、调度算法、API |
| `amd-sys` | 系统工程师 | ROCm/OpenCL 集成、驱动交互 |

## 核心功能

- [ ] GPU 任务队列管理
- [ ] 双卡负载均衡策略
- [ ] 故障自动切换
- [ ] 实时监控面板
- [ ] 任务优先级调度

## 工作流程

1. **PM** 分析需求 → 输出 PRD
2. **Arch** 设计架构 → 输出技术方案
3. **Backend + Sys** 实现核心功能
4. **Leader** 协调集成 → 最终交付
